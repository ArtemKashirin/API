ng-container - избегает создания лишнего HTML-элемента, можно вместо этого использовать HTML-элемент ng-container. ng-container представляет собой HTML-элемент, к которому можно прикрепить структурную директиву, без необходимости создавать дополнительный элемент только для этого, при этом сам ng-container в DOM не появится. ng-container играет важную роль в которой он служит контейнером для динамического внедрения компонента с HTML-шаблоном в другой компонент.
