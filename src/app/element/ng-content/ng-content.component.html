ng-content - позволяет внедрять родительским компонентам код html в дочерние компоненты. Вместо элемента ng-content извне можно будет передать любое содержимое. У ng-content присутствует атрибут select="[selector]" - который выбирает только тот элемент который соответствует заданному селектору "[selector]".
