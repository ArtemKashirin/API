ActivatedRouteSnapshot - класс, содержит информацию о маршруте связанный с компонентом, загруженным в outlet в определенный момент времени. Экземпляр данного класса можно получить из класса ActivatedRoute и его свойства snapshot или из класса Router свойства routerState который вернет класс RouterState и его свойства snapshot который верент класс RouterStateSnapshot и его свойства root который веренет класс ActivatedRoute.
