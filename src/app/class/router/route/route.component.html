Route - интерфейс, представляет собой объект для конфигурации одного единственного МАРШРУТА. Наборы маршрутов собираются в массив типа Routes[]. Массив маршрутов состоящий из Route определяет конфигурацию маршуритизации(Router). Router пытается сапоставить сегменты каждого URL-адреса с каждым Route используя параметры конфигурации определенные в этом объекте Route.
