ViewRef - абстрактный класс который наследуется от базового абстрактного класса ChangeDetectorRef, отражает представления компонента. Компонент класса и связанный с ним шаблон определяют вид. Представление конкретно представлено ViewRef экземпляром, связанным с компонентом. Представление, которое непосредственно принадлежит компоненту, называется основным представлением.
