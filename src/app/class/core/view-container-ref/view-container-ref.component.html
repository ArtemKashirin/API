ViewContainerRef - абстрактный класс, представляет контейнер, в котором к компоненту можно прикрепить одно или несколько представлений. Может содержать основные представления (созданные путем создания экземпляра компонента с помощью createComponent() метода) и встроенные представления (созданные путем создания экземпляра объекта TemplateRef с помощью createEmbeddedView() метода).
