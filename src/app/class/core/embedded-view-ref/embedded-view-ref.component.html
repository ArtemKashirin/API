EmbeddedViewRef - абстрактный класс наследуется от абстрактного класса ViewRef, представляет представление в контейнере представления. На встроенное представление можно ссылаться из компонента, отличного от компонента размещения, шаблон которого определяет его, или он может быть определен независимо с помощью файла TemplateRef.
