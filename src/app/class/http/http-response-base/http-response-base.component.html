HttpResponseBase - базовый абстрактный класс от которого наследуются HttpErrorResponse, HttpHeaderResponse, HttpResponse.

  • headers: HttpHeaders - чтение, все заголовки ответов;
  • status: number - чтение, код состояния ответа;
  • statusText: string - чтение, текстовое описание кода состояния ответа, по умолчанию ОК;
  • url: string | null - чтение, URL-адрес полученного ресурса или null, если он недоступен;
  • ok: boolean - чтение, попадает ли код состояния в диапазон 2xx;
  • type: HttpEventType.Response | HttpEventType.ResponseHeader - чтение, тип ответа, суженный до полного ответа или заголовка.
