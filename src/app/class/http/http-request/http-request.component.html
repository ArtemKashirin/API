HttpRequest - класс, исходящий HTTP-запрос с необязательным типизированным телом. HttpRequest представляет исходящий запрос, включая URL-адрес, метод, заголовки, тело и другие параметры конфигурации запроса. Предполагается, что экземпляры неизменяемы. Чтобы изменить HttpRequest, следует использовать метод clone.
