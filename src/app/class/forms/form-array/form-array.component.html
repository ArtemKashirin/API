FormArray - класс расширяет абстрактный базовый класс AbstractControl, отслеживает состояние массива FormControl. FormArray объединяет значения каждого дочернего элемента FormControl в массив. Он вычисляет свой статус дочерних элементов. Например, если один из элементов FormControl в FormArray недействителен, весь массив становится недействительным.
