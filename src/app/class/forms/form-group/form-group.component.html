FormGroup - класс расширяет абстрактный базовый класс AbstractControl, отслеживает значение и состояние группы FormControl экземпляров. FormGroup объединяет значения каждого дочернего элемента FormControl в один объект с именем каждого элемента FormControl в качестве ключа. Он вычисляет свой статус, проверяя каждый элемент FormControl. Если один из элементов FormControl в группе недействителен, вся группа становится недействительной.
