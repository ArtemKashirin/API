@Component() - декаратор @Component помечает класс как компонент Angular и предоставляет метаданные конфигурации компонента, которые определяют, как этот компонент будет обрабатываться, создаваться и использоваться во время выполения. Компонент должен принадлежать к @NgModule(), чтобы быть доступным для другого компонента или приложения. Чтобы сделать его членом NgModule, укажите его в поле declarations NgModule метаданных.
