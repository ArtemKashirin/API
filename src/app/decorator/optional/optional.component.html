@Optional() - декоратор параметров, который будет использоваться для параметров конструктора, который отмечает параметр как необязательную зависимость. Инфраструктура DI обеспечивает, null если зависимость не обнаружена. Может использоваться вместе с другими декораторами параметров, которые изменяют работу внедрения зависимостей.
