DefaultValueAccessor - является значением по умолчанию ControlValueAccessor, для записи и прослушивания изменений в элеменнтах input. DefaultValueAccessor используется для FormControlDirective, FormControlName и NgModel. DefaultValueAccessor автоматически применяется при использовании директив [formControlName], [formControl], [ngModel] и [ngDefaultControl] неявно или можно явно применить для пользовательских элементов ввода [ngDefaultControl].
