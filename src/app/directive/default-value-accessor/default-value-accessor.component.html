ControlValueAccessor - интерфейс, который действует как мост между формами и кастомным элементом по типу input. При этом, такому элементу(если вдруг нехватает нативного HTML-элемента), можно присвоить значение по умолчанию при помощи селектора: ngDefaultControl.

DefaultValueAccessor - атрибут: ngDefaultControl - аксессор, используется для записи значения и прослушивания изменений в HTML-элементе input с атрибутом type="text". Используется с вышеуказанным HTML-элементом и атрибутом, и директивами formControlName, formControl, ngModel.

CheckboxControlValueAccessor - аксессор, используется автоматически для записи значения и прослушивания изменений в HTML-элементе input с атрибутом type="checkbox". Используется с вышеуказанным HTML-элементом и атрибутом, и директивами formControlName, formControl, ngModel.

NumberValueAccessor - аксессор, используется автоматически для записи значения и прослушивания изменений в HTML-элементе input с атрибутом type="number". Используется с вышеуказанным HTML-элементом и атрибутом, и директивами formControlName, formControl, ngModel.

RadioControlValueAccessor - аксессор, используется автоматически для записи значения и прослушивания изменений в HTML-элементе input с атрибутом type="radio". Используется с вышеуказанным HTML-элементом и атрибутом, и директивами formControlName, formControl, ngModel.

RangeValueAccessor - аксессор, используется автоматически для записи значения и прослушивания изменений в HTML-элементе input с атрибутом type="range". Используется с вышеуказанным HTML-элементом и атрибутом, и директивами formControlName, formControl, ngModel.

SelectControlValueAccessor - аксессор, используется автоматически для записи значения и прослушивания изменений в HTML-элементе select. Используется с вышеуказанным HTML-элементом без атрибута: multiple], и директивами formControlName, formControl, ngModel.

SelectMultipleControlValueAccessor - атрибут: multiple - аксессор, используется для записи значений и прослушивания изменений в HTML-элементах select с множественным выбором. Используется с вышеуказанным HTML-элементом и селектором, и директивами formControlName, formControl, ngModel.
