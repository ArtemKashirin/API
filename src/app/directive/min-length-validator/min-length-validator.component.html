MinLengthValidator - атрибут: minlength - проверка минимальной длинны. Используется с директивами formControlName, formControl, ngModel.

MaxLengthValidator - атрибут: maxlength - проверка максимальной длинны. Используется с директивами formControlName, formControl, ngModel.

MinValidator - атрибут: min - устанавливает минимальное значение для HTML-элемента input с атрибутом type="number". Используется с вышеуказанным HTML-элементом и атрибутом, и директивами formControlName, formControl, ngModel.

MaxValidator - атрибут: max - устанавливает максимальное значение для HTML-элемента input с атрибутом type="number". Используется с вышеуказанным HTML-элементом и атрибутом, и директивами formControlName, formControl, ngModel.

PatternValidator - атрибут: pattern - добавляет проверку вводимого значения регулярного выражения. Используется с директивами formControlName, formControl, ngModel.

RequiredValidator - атрибут: required - добовляет проверку на обязательный ввод значения. Применяется без атрибута: type=checkbox и используется с директивами formControlName, formControl, ngModel.

EmailValidator - атрибут: email - добовляет проверку на ввод email. Применяется к атрибуту: type=email и используется с директивами formControlName, formControl, ngModel.

CheckboxRequiredValidator - атрибут: required - добовляет проверку на обязательный выбор checkbox. Применяется с атрибутом: type=checkbox и используется с директивами formControlName, formControl, ngModel.
