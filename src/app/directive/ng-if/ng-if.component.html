NgIf - структурная директива, которая условно включает шаблон на основе значения выражения, преобразованного в логическое значение. Когда выражение принимает значение true, Angular отображает шаблон, указанный в then предложении, а если значение false или null, Angular отображает шаблон, указанный в необязательном else предложении. Шаблон по умолчанию для else предложения пуст.
