NgControlStatus - автоматически применяется к элементам управления формы, которая устанавливает классы CSS в зависимости от статуса элемента управления. Следующие классы применяются по мере того, как свойства становятся истинными: валидный, недействительный, ожидаемый, нетронутый, грязный, нетронутый, тронутый.
