FormArrayName - синхронизирует вложенный FormArray элемент с элементом DOM. Эта директива предназначена для использования с родительским элементом FormGroupDirective(селектором:) [formGroup]. Он принимает строковое имя вложенного объекта, который FormArray вы хотите связать, и будет искать FormArray зарегистрированный с этим именем в родительском FormGroup экземпляре, в который вы перешли FormGroupDirective.
