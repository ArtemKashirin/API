RouterLinkActive - атрибут: routerLinkActive - отслеживает, активен ли связанный маршрут элемента в настоящий момент, и позволяет добавить один или несколько классов CSS для добавления к элементу, когда связанный маршрут активен и если URL изменяется, класс удаляется. Возможно установить несколько классов в одной директиве через массив или строку.
