RouterOutlet - тег: router-outlet - выступает в качестве заполнителя, на место элемента router-outlet будет рендериться компонент, выбранный для обработки запроса. Также у каждого элемента может быть необязательный атрибут name, для придания каждому маршруту индивидуальности. При помощи атрибута name можно настроить таргетинг на несколько маршрутов в одной RouterLink директиве.
