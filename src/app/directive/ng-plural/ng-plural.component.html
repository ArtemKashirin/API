NgPlural - атрибут: [NgPlural] - добавляет/удаляет поддеревья DOM на основе числового значения. Создан для множественного числа.

NgPluralCase - атрибут: ngPluralCase - создает представление, которое будет добавлено/удалено из родительского, NgPlural когда данное выражение соответствует выражению множественного числа.
